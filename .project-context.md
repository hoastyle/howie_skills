## âš ï¸ Known Pitfalls & Solutions

### Pitfall 1: Parameter Naming Errors
**Problem**: Using `filename` instead of `memory_file_name`
**Solution**: Always reference `docs/MCP_PARAMETER_REFERENCE.md` for correct parameter names

### Pitfall 2: Relative vs Absolute Paths
**Problem**: Magic MCP doesn't accept relative paths
**Solution**: Always use `path.resolve()` to convert to absolute paths

### Pitfall 3: COLD Mode Startup Delay
**Problem**: First call to COLD mode servers takes 5-10 seconds
**Solution**: Be patient on first call; subsequent calls are fast
